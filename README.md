
## Run Locally

Clone the project

```bash
   git clone https://github.com/vinay-colaco/ToDo-List.git
    OR
   git clone git@github.com:vinay-colaco/ToDo-List.git
```

Go to the project directory

```bash
   cd ToDo-List
```

Install dependencies

```bash
   cd client
   npm install
   cd server
   npm install
```

Start the server

```bash
   cd server
   npm start
```
Start the Vite + React

```bash
   cd client
   npm run dev
```
